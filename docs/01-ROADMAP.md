# 产品路线图

路线图是方向性计划，不等同于交付承诺。版本范围变化时应同步更新本文件和 CHANGELOG。

## v0.6 — 数据质量与维护体验

- 完善指标新增、编辑、校验和维护状态反馈
- 明确最新值、前值、涨跌幅/基点变化及变化单位
- 统一展示业务更新时间、数据日期和数据来源
- 增加过期数据、待录入数据和刷新失败的可见提示
- 完善自动更新与手工维护指标的区分
- 为 Staging 准备可重复的测试数据与验收清单

## v0.7 — 交易工作流

- **Narrative**：形成简洁的当日市场叙事
- **Driver**：标注推动市场变化的关键因素
- **Watchlist**：允许用户保存重点指标组合
- **Alerts**：支持阈值、涨跌幅和数据过期提醒
- 增强宏观日历筛选、重要性和实际值跟踪
- 支持按用户关注顺序组织首页信息

## v1.0 — 可持续的市场智能平台

- **AI 分析**：基于可追溯数据生成市场摘要、异常解释和风险提示
- 关联指标、事件、Narrative 与 Driver，形成上下文分析
- 用户权限、审计记录与团队协作
- 更可靠的数据任务调度、失败重试和质量监控
- 发布稳定的 API 契约和数据版本策略
- 建立可观测性、备份恢复和正式运行指标

## 暂不承诺

- 自动交易或下单
- 未经授权的商业数据分发
- 用生成内容替代原始数据来源或专业判断

未来 Admin 与自动数据源仅处于设计阶段，分别见 [ADMIN](future/ADMIN.md) 和 [AUTO_DATA](future/AUTO_DATA.md)。
