## 不采用的捷径

- 不把网页能抓取等同于稳定 API。
- 不用搜索结果或截图作为自动化真值源。
- 不以 CCXT 解决非加密资产。
- 不为 pandas-datareader 单独引入 Python；FRED 可由 Node 直接读取。
- 不用 Alpha Vantage 月度 Copper 替代每日伦铜。
- 不用 service key 或浏览器直连供应商。
